About DirectLineAPI

This is a partial implementation from official Microsoft's Azure™ DirectLine technology ported to attend the requirements to internally develop chat bots

You can see the published API's throu the API Primitives menu. Well it's not useful to be used directly. You'll need a BOT client and a BOT service as required and WELL explained in Microsoft Azure's BotFramework technology.

Disclaimer

DirectLineAPI serves for the purpose of allow BOT clients to communicate with the Developed BOT services using Microsoft BotFramework. It's a partial implementation but it's full function for many of the purposes.

Although it is safe to use. I advise some discretion to use it and be wary of the access granted to this tool. Keep your generated tokens up to date and safely stored.
